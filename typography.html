Typography

Typography helps create the structure and heirarchy of a page. This helps users read, absorb and comprehend the content. We use Roboto for headings and Open Sans for body and UI elements.

These typefaces were choosen for their pairing, extensive range of weights and similarity to Helvetica.

Title 1

Usage

Do

  • Top of the the page only
  • Commonly used on landing pages - home page
  • Sentance case only

Do not

  • Use more than once on the page
  • Exceed 140 characters
  • Add hyphens to line wrap text

Example

<h1 class="title-1">Title-1</h1>
Great value health insurance

CSS

			    		.title-1 {

			    		font-family: roboto;

			    		font-size: 54px;

			    		font-weight: 300;

			    		letter-spacing -2px;

			    		line-height: 60px;

			    		margin-bottom:

			    		};

Copy

Usage

Do

  • For paragraphs and general elements
  • Use minimum of 20px margin bottom between paragraphs
  • Prefer left aligned

Do not

  • use a font size less than 14px
  • add hyphens to line wrap text
  • exceed six columns on a desktop
  • center large blocks of text
  • justify text

Example:

We'll recognise any waiting periods already served with your current fund, so you can claim straight away. See our change your health insurance page. Any existing limits used with your current fund will carry across to nib.

HTML

<p class="copy">Copy</p>


CSS

			    		.copy {

		    			color: #444444;
						font-family: open-sans;

						font-size: 14px;

						font-weight: 400;

						line-height: 20px;

			    		margin-bottom:

			    		};
